# CH32 Development Board

This repository contains hardware design files and documentation for a development board based on the CH32V30x series of microcontrollers. The board features USB-FS and UART communication interfaces, as well as input current monitoring. It is intended as a test and development platform for projects using the CH32V30x family.

Documentation is available in the [docs](./docs) folder, including:
- [Component datasheets and specifications](./docs/01_specs/)
- [Design calculations](./docs/02_calcs/)

For compatible firmware, see the related firmware repository
➡️ [ch32-devboard-firmware](https://github.com/samvangysegem/ch32-devboard-firmware)
